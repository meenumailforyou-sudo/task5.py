##Data Cleaning & Reporting Automation
This project automates data cleaning and reporting workflows using Python. It helps improve data quality by handling missing values, removing duplicate records, and generating automated reports and visual summaries.

##Features
Data loading using Pandas
Removal of duplicate records
Handling of missing values
Automated report generation
Visual summary creation using Matplotlib
Export of cleaned dataset
##Technologies Used
Python
Pandas
Matplotlib
VS Code
##Project Workflow
Load the employee dataset.
Identify and remove duplicate records.
Handle missing values in the dataset.
Generate a summary report.
Create visualizations for numerical data.
Save the cleaned dataset and reports.
